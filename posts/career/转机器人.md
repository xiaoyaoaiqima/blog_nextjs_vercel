你做**Agent开发**，转**家用机器人**是**天然高匹配、最平滑的转型路径**之一——家用机器人的“大脑”就是**具身智能Agent**，你的规划、决策、工具调用、多轮交互、反思循环等能力，正是行业最缺的核心技能。下面给你一套**可直接执行的转型方案**：岗位定位、技能补全、简历/项目包装、投递路径、薪资预期，一步到位。

## 一、先明确：你最适合的家用机器人岗位（直接对标）
### 1. 首选岗位：**家庭机器人AI Agent/具身智能工程师**（最匹配、最容易上岸）
- 核心职责（和你现在做的高度重合）：
  - 设计机器人**任务规划、决策引擎、工具调用（Function Calling）、反思循环（Agentic Loop）**
  - 实现**自然语言指令 → 任务拆解 → 动作执行 → 结果反馈**的全链路
  - 搭建**多模态交互**（语音+视觉+触觉）的Agent系统
  - 优化**推理效率、上下文管理、长程任务可靠性**
  - 与**感知（SLAM/视觉）、运动控制、硬件**团队协同，把数字Agent落地到物理机器人
- 招聘关键词：**家庭机器人Agent、具身智能、任务规划、行为树、状态机、大模型+机器人**
- 代表公司：可以科技、追觅、科沃斯、iRobot、Vbot维他动力、石头科技、云鲸、宇树科技等

### 2. 次选岗位（平滑过渡）
- **机器人软件/系统工程师**：负责Agent与ROS/ROS2、硬件驱动、传感器的集成
- **机器人规划算法工程师**：做路径规划、任务调度、行为决策（A*/RRT*/DWA/MPC/行为树）
- **多模态（VLM）算法工程师**：视觉-语言-动作模型，适配家庭场景

## 二、技能补全：从“纯软件Agent”到“家用机器人Agent”（3个月速成）
### 1. 必须掌握（你已有+快速补）
#### （1）你已有的核心优势（直接复用）
- **大模型+Agent框架**：LangChain、AutoGen、CrewAI、Function Calling、Agentic Loop、反思机制
- **任务规划/编排**：状态机、行为树、工作流、长程任务拆解
- **多轮交互/上下文管理**：记忆、工具调用、错误恢复、重试机制
- **Python/PyTorch**、推理优化（量化/蒸馏/KV Cache）、RAG、微调（SFT/RLHF/DPO）

#### （2）必须补的机器人领域知识（3个月优先级）
1. **机器人操作系统（ROS/ROS2）**：核心！家用机器人标配
   - 掌握：节点/话题/服务/动作、rviz可视化、Gazebo仿真、moveit!（机械臂）、导航栈（Nav2）
   - 学习路径：ROS2 Humble → 官方教程 + 动手做一个“机器人导航+语音指令”Demo
2. **机器人基础（运动/感知/控制）**
   - 运动学/动力学：正逆运动学、底盘控制、机械臂控制、PID/MPC基础
   - 感知与定位：SLAM、语义建图、避障（ToF/激光/视觉）、传感器融合（IMU/相机/雷达）
   - 路径规划：A*/RRT*/DWA、全局+局部规划、动态避障
3. **具身智能核心**
   - 行为树/状态机：用于机器人任务决策（家庭场景：清洁/跟随/取物/交互）
   - 仿真环境：Isaac Sim、MuJoCo、PyBullet、Gazebo（训练+验证）
   - 物理交互：力控、抓取、操作、Sim-to-Real迁移
4. **家庭场景知识**
   - 家庭环境特点：动态障碍物、复杂布局、用户指令模糊、安全约束
   - 典型任务：清洁、陪伴、看护、取物、问答、智能家居控制

### 2. 工具栈（必须会）
- 语言：**Python（主力）、C++（加分，性能/嵌入式）**
- 框架：ROS2、PyTorch/TensorFlow、LangChain、AutoGen、行为树库（BehaviorTree.CPP）
- 仿真：Isaac Sim、MuJoCo、Gazebo、PyBullet
- 部署：Linux、Docker、边缘推理（TensorRT/ONNX）、机器人硬件接口（CAN/串口/TCP）

## 三、简历/项目包装：把Agent经验“翻译”成机器人经验（最关键一步）
### 1. 简历关键词替换（让HR一眼看到匹配度）
- 把“**AI Agent开发**” → “**家庭机器人具身智能Agent/任务规划系统开发**”
- 把“**任务编排/工具调用**” → “**机器人任务规划/行为决策/Function Calling落地**”
- 把“**多轮对话/上下文**” → “**机器人多模态交互/长程任务记忆/用户意图理解**”
- 把“**推理优化/性能**” → “**机器人端侧推理优化/低延迟响应/资源效率提升**”

### 2. 做1-2个家用机器人Agent项目（3个月内完成，直接写进简历）
#### 项目1：家庭清洁机器人Agent（最通用、最加分）
- 功能：语音指令（如“打扫客厅、避开地毯、1小时后回充”）→ 任务拆解 → 路径规划 → 执行 → 反馈
- 技术栈：ROS2 + Nav2 + 大模型（GPT-4o/Qwen）+ 行为树 + 语音ASR/TTS
- 亮点：**Agent规划+机器人导航+物理执行**的全链路，体现“数字→物理”落地能力

#### 项目2：家庭陪伴机器人Agent（差异化）
- 功能：多模态交互（语音+视觉）+ 情感识别 + 任务调度（提醒/看护/娱乐）+ 智能家居控制
- 技术栈：VLM（Qwen-VL）+ Agent + 行为树 + 智能家居API（米家/HomeKit）
- 亮点：体现**具身智能、多模态、家庭场景适配**能力

#### 项目3：开源贡献/复现
- 复现：具身智能论文（如RT-2、PaLM-E、Diffusion Policy）
- 贡献：家用机器人开源项目（如OpenManipulator、TurtleBot4、家用清洁机器人开源方案）

## 四、投递路径：精准找岗、高效上岸
### 1. 目标公司梯队（按优先级）
#### （1）第一梯队：家用机器人头部（最稳、薪资高）
- 清洁机器人：**科沃斯、石头科技、追觅、云鲸、iRobot**（全球龙头）
- 陪伴/全能机器人：**可以科技、Vbot维他动力、优必选、宇树科技**
- 大厂机器人部门：小米机器人、华为终端机器人、百度智能驾驶（家用方向）

#### （2）第二梯队：成长型/具身智能公司（机会多、匹配度高）
- 专注具身智能：积加科技、心言机器人、卧安机器人（OneRobotics）
- 创业公司：家用服务机器人、AI+机器人初创（Agent岗位多）

### 2. 投递渠道
- 招聘平台：BOSS直聘、智联、猎聘（搜关键词：**家用机器人 Agent、具身智能、任务规划**）
- 公司官网/公众号：直接投研发岗（AI/算法/软件）
- 开源社区：GitHub（家用机器人项目）、ROS社区、具身智能社群
- 内推：LinkedIn、脉脉、技术社群（最有效）

### 3. 面试准备（高频考点）
- Agent+机器人结合：**如何把大模型指令转化为机器人动作？**（任务拆解+行为树+运动控制）
- 家庭场景挑战：**动态避障、用户指令模糊、安全约束、低延迟、资源受限**
- 技术细节：ROS2、行为树、路径规划、仿真、推理优化、多模态融合
- 项目深挖：你的Agent如何解决机器人的**可靠性、鲁棒性、Sim-to-Real**问题
