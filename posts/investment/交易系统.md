## 操作模式

### 复盘
1、大盘指数形态

2、题材板块，是否有主线，潜在主线。

```
你作为一个短线高手。结合重要/最新的科技/政治/经济新闻。
来分析**当前a股市场**主线题材。目前市场可能有多个题材，处于不同时期，可能是萌芽，高潮，退潮，反包。
你可以从题材的想象空间，涉及的资金量，板块效应入手分析。
if 当前市场情绪好（高连板，指数共振）
then 目标是找到高潮阶段题材的龙头股（高连板，情绪标的）和容量中军（机构票，市值大）。
else 市场情绪一般（低连板，市场震荡）
then 目标是博弈轮动修复的预期，即处在萌芽/反包阶段的龙头，中军。
另外，未来 5-10 年 A 股投资将围绕 **“科技自主、能源转型、制造升级、民生改善”四大主线展开。我主观建议跟踪的题材：半导体 + AI、新能源汽车 + 储能、创新药 + 医疗器械、AGI + 具身智能、商业航天、合成生物学、网络安全 + 数据要素、低空经济。
当然，你重点关注当下热点。

```

### 预案
1、明日需要关注的标的，预期，相应操作。
例如：三峡旅游，预期上涨，持仓。如果下跌，减仓。

## 早盘准备
1、对美股夜盘的分析，元宝、豆包。

提示词：
```
请分析美股整体情况，内容必须包含以下5点并给出对**今日A股**的影响总结：
1. 美股三大指数收盘表现，领涨/领跌板块、涨幅居前板块及个股，简要分析原因；
2. 分析七巨头和其他关键赛道龙头的走势及原因；
3. 中概股表现，分析驱动因素；
4. 当日出现明显异动（大涨/大跌）的重点股票及原因；
5. 知名机构或知名投资者发布的重要研报、持仓变动或观点；
最后给出明确总结：昨夜美股对**今日A股**的情绪与板块影响。
```

2、集合竞价，看昨日题材是否可持续。
