▎Q19: Agent怎么调用工具？
Function Calling / ReAct / Tool Use
流程：LLM决策 → 提取参数 → 执行 → 返回结果 → 生成回答
▎Q20: 工具调用失败怎么办？
超时→重试 | 参数错误→校验 | 权限不足→降级 | 资源耗尽→排队
核心：重试策略 + 降级方案
▎Q21: 怎么做Agent评估？
准确性（正确率）、效率（步数/时间）、鲁棒性（错误率）、有用性（满意度）
工具：LangSmith / RAGAs / Prometheus
▎Q22: Agent安全怎么保证？
威胁：Prompt注入、工具滥用、数据泄露
防护：输入过滤、权限控制、脱敏、沙箱隔离
▎Q23: Agent性能怎么优化？
延迟→缓存+异步 | 成本→Token优化 | 工具→并行执行
架构：流式输出、批处理、多级缓存
